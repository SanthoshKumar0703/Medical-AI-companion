# 🩺 Medical AI Companion

An intelligent AI-powered healthcare companion developed using **Java, JSP, Servlets, MySQL, HTML, CSS, and JavaScript**. The application helps users perform basic symptom analysis, receive AI-generated health guidance, and maintain a history of previous health assessments through a simple and user-friendly interface.

---

## 📌 Project Overview

MedAI Companion is a web-based healthcare assistant designed to provide preliminary health insights based on user-reported symptoms. It aims to improve accessibility to basic health information while offering a clean and responsive user experience.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YourUsername/MedAI-Companion.git
```

### 2. Import the Project

Open the project in NetBeans IDE.

### 3. Configure Database

Create a MySQL database and import the required tables.

### 4. Update Database Connection

Modify the JDBC connection details according to your local MySQL configuration.

### 5. Deploy

Run the project using Apache Tomcat.

Open:

```
http://localhost:8080/MedAI_companion/
```
